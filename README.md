# 🏥 Oilaviy sog'liq kuzatuv

Oila a'zolarining tibbiy tahlil natijalari va shifokor ko'riklari uchun shaxsiy kuzatuv tizimi.

## Imkoniyatlar

- 👨‍👩‍👧‍👦 Oila a'zolarini qo'shish va boshqarish
- 🧪 Tahlil natijalarini kiritish va kuzatib borish
- 🩺 Shifokor ko'riklari va tashxislarni saqlash
- 📊 Vaqt bo'yicha grafiklar bilan natijalarni solishtirish
- ⚡ AI yordamida tahlil va tavsiyalar olish (ixtiyoriy)
- 📱 Telefon va kompyuterda ishlaydi
- 🌙 Qorong'u/yorug' rejim avtomatik

## GitHub Pages'ga joylashtirish

### 1-qadam: GitHub'da yangi repo yarating

1. [github.com/new](https://github.com/new) sahifasiga kiring
2. Repository nomi: `sogliq` (yoki boshqa nom)
3. **Public** ni tanlang
4. "Create repository" tugmasini bosing

### 2-qadam: Faylni yuklang

1. Yangi repo sahifasida **"uploading an existing file"** havolasini bosing
2. `index.html` faylini sudrab olib tashlang (drag & drop)
3. "Commit changes" tugmasini bosing

### 3-qadam: GitHub Pages'ni yoqing

1. Repo sahifasida **Settings** → **Pages** bo'limiga kiring
2. **Source** qismida **"Deploy from a branch"** ni tanlang
3. **Branch**: `main`, papka: `/ (root)` → **Save**
4. 1-2 daqiqa kuting

### 4-qadam: Tayyor!

Sayt manzili: `https://SIZNING_USERNAME.github.io/sogliq/`

Bu havolani oila a'zolaringizga yuboring — har bir kishi o'z telefonida o'z ma'lumotlarini kiritadi.

## AI funksiyasini sozlash (ixtiyoriy)

AI tahlil funksiyasi uchun Anthropic API kaliti kerak:

1. [console.anthropic.com](https://console.anthropic.com) saytida ro'yxatdan o'ting
2. **Settings** → **API Keys** → **Create Key**
3. Ilovada ⚡ tugmasini bosib, kalitni kiriting

> **Eslatma:** API kalit faqat sizning brauzeringizda saqlanadi. Har bir oila a'zosi o'z kalitini kiritishi kerak (yoki AI'siz foydalanishi mumkin).

## Ma'lumotlar xavfsizligi

- Barcha ma'lumotlar faqat brauzerning `localStorage`'ida saqlanadi
- Hech qanday server yoki bulutga yuborilmaydi (AI dan tashqari)
- Har bir kishi faqat o'z ma'lumotlarini ko'radi
- Brauzer ma'lumotlarini tozalasangiz, ma'lumotlar o'chadi
